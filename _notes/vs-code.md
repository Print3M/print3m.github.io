## 1. Linux

### 1.1. Shortcuts

```text
ctrl + shft + p                     # Open command line
> settings.json                     # Settings file
> theme                             # Select theme

alt + arrow                         # Move line up/down
ctrl + shft + k                     # Delete line

ctrl + shft + a                     # Block comment
ctrl + /                            # Line comment

ctrl + 0                            # Fold all sections
```
